# Security Policy

## Supported versions

dpilot is pre-1.0, so only the latest release receives security fixes.

| Version | Supported |
| --- | --- |
| latest (`v0.1.x`) | :white_check_mark: |
| older | :x: |

## Reporting a vulnerability

Please report security issues **privately** — do not open a public issue.

Use the **"Report a vulnerability"** button on the
[Security tab](https://github.com/haraldpdl/dpilot/security) (GitHub private
vulnerability reporting). I'll acknowledge the report, work on a fix, and publish
an advisory once it's resolved — with credit if you'd like.
